Difference Between Local-Storage and Session Storage ?

Ans:The read-only sessionStorage property accesses a session Storage object for the current origin. sessionStorage is similar to localStorage ; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.

What is difference between local and global scope in JavaScript?

Ans:JavaScript variables have only two scopes. Global Variables − A global variable has a global scope which means it can be defined anywhere in your JavaScript code. Local Variables − A local variable will be visible only within a function where it is defined. Function parameters are always local to that function..

HOW Event loop Works on Javascripts?

Ans:The Event Loop has one simple job — to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.

The reasons of getting undefined?

Ans:

01. A variable that has not been assigned a value is of type undefined

02. A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value.

03. A function returns undefined if a value was not returned.

04. A property which is not exist on an object,if we try to access this property then we get undefined.
